#  Django Todo App

A full-stack Todo application built using **Django, HTMX, Tailwind CSS, and MySQL**.  
This project allows users to register, login, and manage their todos dynamically without page reload.

---

##  Features

- 🔐 User Registration
- 🔑 User Login & Logout
- ➕ Add Todo
- ✏️ Edit Todo
- ❌ Delete Todo
- ✅ Mark Todo as Complete / Incomplete
- 🎯 Priority Selection (Low / Medium / High)
- 👤 User-specific Todo Management
- ⚡ Dynamic UI updates using HTMX (No page reload)
- 🛠 Django Admin Panel

---

##  Tech Stack

- **Backend**: Django
- **Frontend**: Django Templates, HTMX, Tailwind CSS
- **Database**: MySQL (Development), PostgreSQL/SQLite (Deployment)
- **Authentication**: Django built-in authentication
- **Deployment**: Render
